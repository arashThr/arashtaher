Lightning Round

Opinions, tools & hard-won lessons from working on Pensive

Arash Taher · Volvo Cars

Agentic Foundation · April 2026

01

Have a personal project to test & learn

Don't just read docs. Build something real that's yours.
Break it, fix it, ship it. That's how you actually learn.

02

Heavy on interaction? Start with the interface

Think thoroughly about the UX first.
Use Google Stitch to prototype fast.

03

Heavy on logic? Start with tests

Write the tests first. Let them guide your design.
CI is important. automate from day one.

04

RAGs are good, but maybe wikis are better

Not everything needs vector search.
Sometimes a well-organized wiki gives you more signal, less noise.

05

Local models are still not great

Phi-4 was underwhelming…
But Gemma 4 might have changed the game. Keep testing.

06

Voice generation

Kokoro (Local)
Works, but resource-intensive
ElevenLabs
Great quality, but expensive
Google Vertex AI
Cheap, but the UI sucks
07

React has won by being the default

But don't let that close your eyes to alternatives.
Svelte, Solid, HTMX... find what works best for you.

08

Curate your repo of base apps

Build a collection of starter templates.
Then use agents to build on top.

09

Telegram is awesome

Bots provide an amazing interface for chats
it's the most developer-friendly messaging platform out there.

10

Don't worry, let it cook: MCP

If it's not pretty or elegant, don't touch it. Wait for something better
Give it time.

11

Linux is easier than AWS… and cheaper

A $5 VPS can do a lot.
Knowing Linux is cool again

12

Context is everything

Obsessively collect and document.

  • Memos: capture thoughts instantly
  • Emacs org mode: integrate with your workflow
🎤

That's it. Go build.
